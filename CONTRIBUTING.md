# Contributing to PartySpot

Thank you for your interest in contributing to PartySpot! We welcome contributions from the community to help make this the best nightlife platform in Cologne.

## How to Contribute

1. **Bug Reports**: If you find a bug, please create an issue on GitHub describing the problem and steps to reproduce.
2. **Feature Requests**: Have an idea for a new feature? Open an issue!
3. **Pull Requests**:
   - Fork the repository.
   - Create a new branch for your feature or bugfix.
   - Ensure your code follows the existing style and architecture (Clean Architecture).
   - Run `npm run lint` and `npm run build` before submitting.
   - Open a pull request with a clear description of your changes.
